# Soirée métiers 2025-2026

A one page website for the 2025-2026 edtition of the infamous "Soirée Métiers Multimédia".

Runs on [Eleventy](https://www.11ty.dev/), [Lightning CSS](https://lightningcss.dev/) and [esbuild](https://esbuild.github.io/).

## Install

- Clone or download repository
- `npm i` to install dependencies
- `npm run dev` to run a development server
- `npm run build` to build the project
